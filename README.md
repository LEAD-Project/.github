# LEAD

### 關於我們
本組織於成立於 **國立臺北科技大學 (NTUT)**，專注於 **人工智慧 (Artificial Intelligence)** 與 **自動化軟體工程 (Automated Software Engineering)** 的跨領域研究。

本組織作為存放實驗數據、原始程式碼與研究成果的核心平台。

> **注意：** 具體專案細節目前處於開發階段並暫時保密。公開的 Repository 將在專案完成後釋出。
